## API 参考

### 请求

```bash
POST https://api.ruxa.ai/api/v1/tasks/create
```

### 请求头

```bash
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
```

### 请求体参数

| 参数                   | 类型    | 必填 | 描述                                                      |
| ---------------------- | ------- | ---- | --------------------------------------------------------- |
| `model`                | string  | 是   | 模型名称：`sora-2` 或 `sora-2-hd`                         |
| `prompt`               | string  | 是   | 视频描述提示词                                            |
| `seconds`              | string  | 否   | 视频时长：`10`、`15`、`25`（仅 pro 支持）                 |
| `input_reference`      | file    | 否   | 输入参考图像                                              |
| `size`                 | string  | 否   | 视频尺寸（见下方说明）                                    |
| `watermark`            | boolean | 否   | 是否需要水印，默认无水印                                  |
| `private`              | boolean | 否   | 是否隐藏视频，true 时视频不会发布且无法 remix，默认 false |
| `character_url`        | string  | 否   | 创建角色需要的视频链接（视频中不能出现真人）              |
| `character_timestamps` | string  | 否   | 视频角色出现的秒数范围，格式 `{start},{end}`，范围 1~3 秒 |
| `metadata`             | string  | 否   | 元数据                                                    |
| `character_from_task`  | string  | 否   | 已完成的任务 ID，可根据已生成的任务创建角色               |
| `character_create`     | boolean | 否   | 创建视频完成后，自动根据生成的视频创建角色                |
| `style`                | string  | 否   | 风格：`winter`（冬天）、`comic`（漫画）、`news`（新闻）   |

### Size 尺寸说明

| 值          | 描述     | 备注              |
| ----------- | -------- | ----------------- |
| `1024x1792` | 高清竖屏 | 仅 sora-2-hd 可用 |
| `1792x1024` | 高清横屏 | 仅 sora-2-hd 可用 |
| `720x1280`  | 标清竖屏 |                   |
| `1280x720`  | 标清横屏 |                   |

<Note>
  宽 > 高 为横屏，宽 < 高 为竖屏
</Note>

### 请求示例

```json
{
  "model": "sora-2",
  "prompt": "牛马起飞",
  "seconds": "10",
  "size": "1280x720",
  "watermark": false,
  "private": false
}
```

### 带角色创建的请求示例

```json
{
  "model": "sora-2",
  "prompt": "角色在草地上奔跑",
  "seconds": "10",
  "size": "1280x720",
  "character_url": "https://example.com/character-video.mp4",
  "character_timestamps": "0,3",
  "character_create": true
}
```

### 带风格的请求示例

```json
{
  "model": "sora-2",
  "prompt": "城市街道场景",
  "seconds": "10",
  "size": "1280x720",
  "style": "comic"
}
```

### 响应示例

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "taskId": "task_sora-2_1765184045509"
  }
}
```

### 响应状态码

| 状态码 | 描述                            |
| ------ | ------------------------------- |
| 200    | 成功                            |
| 401    | 未授权 - 身份验证凭据缺失或无效 |
| 402    | 积分不足                        |
| 404    | 未找到                          |
| 422    | 验证错误 - 请求参数未通过校验   |
| 429    | 速率限制                        |
| 500    | 服务器错误                      |
| 501    | 生成失败                        |
