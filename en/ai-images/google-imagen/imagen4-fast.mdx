## Introduction

Google Imagen4 Fast is the speed-optimized version of Imagen4, delivering high-quality images with significantly faster generation times. Ideal for scenarios requiring quick results.

## Request Example

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.kie.ai/api/v1/jobs/createTask \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "google/imagen4-fast",
    "callBackUrl": "https://your-domain.com/api/callback",
    "input": {
      "prompt": "A cinematic city nightscape, neon lights reflecting on rain-soaked streets, cyberpunk style, high contrast",
      "negative_prompt": "",
      "aspect_ratio": "16:9",
      "num_images": "1"
    }
  }'
```

</CodeGroup>

## Parameters

| Parameter             | Type   | Required | Description                                    |
| --------------------- | ------ | -------- | ---------------------------------------------- |
| model                 | string | Yes      | Model name, fixed as `google/imagen4-fast`     |
| callBackUrl           | string | No       | Callback URL when task completes               |
| input.prompt          | string | Yes      | Text description for image generation          |
| input.negative_prompt | string | No       | Negative prompt, describes unwanted content    |
| input.aspect_ratio    | string | No       | Image aspect ratio, e.g. `1:1`, `16:9`, `9:16` |
| input.num_images      | string | No       | Number of images to generate                   |

## Comparison with Standard Version

| Feature  | Imagen4           | Imagen4 Fast                    |
| -------- | ----------------- | ------------------------------- |
| Speed    | Standard          | Faster                          |
| Quality  | Highest           | High                            |
| Use Case | Fine art creation | Quick preview, batch generation |
