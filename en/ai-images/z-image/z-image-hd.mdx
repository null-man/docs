## Overview

Z-Image HD is the high-definition version of the Z-Image series, designed for professional scenarios requiring 4K resolution output. It maintains the core advantages of Z-Image while providing higher resolution and finer detail rendering.

## Request Parameters

<ParamField body="model" type="string" required>
  Model name, fixed as `z-image-hd`
</ParamField>

<ParamField body="prompt" type="string" required>
  Text description for image generation, supports Chinese and English
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  Image aspect ratio options: - `1:1` - Square (default) - `16:9` - Landscape
  widescreen - `9:16` - Portrait vertical - `4:3` - Standard landscape - `3:4` -
  Standard portrait - `3:2` - Photo landscape - `2:3` - Photo portrait - `21:9`
  - Ultra-wide
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  Output resolution options: - `1K` - 1024px - `2K` - 2048px (default) - `4K` -
  4096px
</ParamField>

<ParamField body="n" type="integer" default="1">
  Number of images to generate, range 1-4
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://your-api-domain.com/v1/images/generations" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "z-image-hd",
    "prompt": "Professional product photography: a premium skincare bottle on a marble surface, soft natural light from the side, blurred background",
    "aspect_ratio": "4:3",
    "resolution": "4K",
    "n": 1
  }'
```

```python Python
import requests

response = requests.post(
    "https://your-api-domain.com/v1/images/generations",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "z-image-hd",
        "prompt": "Professional product photography: a premium skincare bottle on a marble surface, soft natural light from the side, blurred background",
        "aspect_ratio": "4:3",
        "resolution": "4K",
        "n": 1
    }
)

result = response.json()
print(result)
```

```javascript JavaScript
const response = await fetch(
  "https://your-api-domain.com/v1/images/generations",
  {
    method: "POST",
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      model: "z-image-hd",
      prompt:
        "Professional product photography: a premium skincare bottle on a marble surface, soft natural light from the side, blurred background",
      aspect_ratio: "4:3",
      resolution: "4K",
      n: 1,
    }),
  }
);

const result = await response.json();
console.log(result);
```

## Z-Image Series Comparison

| Feature          | Z-Image  | Z-Image Turbo   | Z-Image HD          |
| ---------------- | -------- | --------------- | ------------------- |
| Max Resolution   | 2K       | 2K              | 4K                  |
| Speed            | Standard | Ultra-fast      | Slower              |
| Detail Precision | High     | High            | Very High           |
| Use Case         | General  | Rapid iteration | Professional output |
| Price            | Standard | Standard        | Higher              |

<Warning>
  4K resolution generation takes longer. We recommend using callback
  notifications instead of polling in production environments.
</Warning>
