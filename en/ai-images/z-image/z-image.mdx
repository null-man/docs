## Overview

Z-Image is an efficient image generation model from Alibaba's Tongyi Qianwen team, built on a 6B parameter single-stream diffusion Transformer architecture. It generates high-quality realistic images with bilingual text rendering support.

## Request Parameters

<ParamField body="model" type="string" required>
  Model name, fixed as `z-image`
</ParamField>

<ParamField body="prompt" type="string" required>
  Text description for image generation, supports Chinese and English
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  Image aspect ratio options: - `1:1` - Square (default) - `16:9` - Landscape
  widescreen - `9:16` - Portrait vertical - `4:3` - Standard landscape - `3:4` -
  Standard portrait - `3:2` - Photo landscape - `2:3` - Photo portrait
</ParamField>

<ParamField body="n" type="integer" default="1">
  Number of images to generate, range 1-4
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://your-api-domain.com/v1/images/generations" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "z-image",
    "prompt": "An orange cat sitting on a windowsill, sunlight streaming through the window, warm afternoon atmosphere",
    "aspect_ratio": "1:1",
    "n": 1
  }'
```

```python Python
import requests

response = requests.post(
    "https://your-api-domain.com/v1/images/generations",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "z-image",
        "prompt": "An orange cat sitting on a windowsill, sunlight streaming through the window, warm afternoon atmosphere",
        "aspect_ratio": "1:1",
        "n": 1
    }
)

result = response.json()
print(result)
```


## Response Example

### Sync Response (Returns Task ID)

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "taskId": "task_z-image_1765174270120"
  }
}
```

### Async Result (Via Task Query)

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "taskId": "task_z-image_1765174270120",
    "state": "success",
    "resultJson": "{\"resultUrls\":[\"https://cdn.example.com/generated/image1.png\"]}"
  }
}
```

## Response Status Codes

| Code | Description                                                  |
| ---- | ------------------------------------------------------------ |
| 200  | Success - Request processed successfully                     |
| 401  | Unauthorized - Authentication credentials missing or invalid |
| 402  | Insufficient Credits - Account lacks credits for operation   |
| 422  | Validation Error - Request parameters failed validation      |
| 429  | Rate Limited - Request limit exceeded                        |
| 500  | Server Error - Unexpected error during processing            |
| 501  | Generation Failed - Content generation task failed           |
