## Overview

Z-Image-Turbo is a distilled accelerated version of Z-Image, achieving comparable or better results than leading competitors with only 8 function evaluations (NFE). It delivers sub-second inference latency on enterprise H800 GPUs while also running smoothly on consumer devices with 16GB VRAM.

## Request Parameters

<ParamField body="model" type="string" required>
  Model name, fixed as `z-image-turbo`
</ParamField>

<ParamField body="prompt" type="string" required>
  Text description for image generation, supports Chinese and English
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  Image aspect ratio options: - `1:1` - Square (default) - `16:9` - Landscape
  widescreen - `9:16` - Portrait vertical - `4:3` - Standard landscape - `3:4` -
  Standard portrait
</ParamField>

<ParamField body="n" type="integer" default="1">
  Number of images to generate, range 1-4
</ParamField>

## Request Example

<CodeGroup>

```bash cURL
curl -X POST "https://your-api-domain.com/v1/images/generations" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "z-image-turbo",
    "prompt": "A futuristic cityscape at sunset, neon lights reflecting on wet streets",
    "aspect_ratio": "16:9",
    "n": 1
  }'
```

```python Python
import requests

response = requests.post(
    "https://your-api-domain.com/v1/images/generations",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "z-image-turbo",
        "prompt": "A futuristic cityscape at sunset, neon lights reflecting on wet streets",
        "aspect_ratio": "16:9",
        "n": 1
    }
)

result = response.json()
print(result)
```

```javascript JavaScript
const response = await fetch(
  "https://your-api-domain.com/v1/images/generations",

    method: "POST",

      n: 1,
    }),
  }

const result = await response.json();
console.log(result);
```
