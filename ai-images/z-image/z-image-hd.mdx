## 概述

Z-Image HD 是 Z-Image 系列的高清版本，专为需要 4K 分辨率输出的专业场景设计。在保持 Z-Image 核心优势的同时，提供更高分辨率和更精细的细节表现。

## 请求参数

<ParamField body="model" type="string" required>
  模型名称，固定为 `z-image-hd`
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本描述，支持中英文
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  图像宽高比，可选值： - `1:1` - 正方形（默认） - `16:9` - 横向宽屏 - `9:16` -
  纵向竖屏 - `4:3` - 标准横向 - `3:4` - 标准纵向 - `3:2` - 照片横向 - `2:3` -
  照片纵向 - `21:9` - 超宽屏
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  输出分辨率，可选值： - `1K` - 1024px - `2K` - 2048px（默认） - `4K` - 4096px
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成图像数量，范围 1-4
</ParamField>

## 请求示例

<CodeGroup>

```bash cURL
curl -X POST "https://your-api-domain.com/v1/images/generations" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "z-image-hd",
    "prompt": "专业产品摄影：一瓶高端护肤品放在大理石台面上，柔和的自然光从侧面照射，背景虚化",
    "aspect_ratio": "4:3",
    "resolution": "4K",
    "n": 1
  }'
```

```python Python
import requests

response = requests.post(
    "https://your-api-domain.com/v1/images/generations",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "model": "z-image-hd",
        "prompt": "专业产品摄影：一瓶高端护肤品放在大理石台面上，柔和的自然光从侧面照射，背景虚化",
        "aspect_ratio": "4:3",
        "resolution": "4K",
        "n": 1
    }
)

result = response.json()
print(result)
```

## Z-Image 系列对比

| 细节精度   | 高      | 高            | 极高       |

## 最佳实践

<Tip>
  **适用场景**： - 印刷品设计（海报、画册、包装） - 大屏展示和广告牌 -
  电商产品主图 - 需要放大查看细节的场景
</Tip>

<Warning>4K 分辨率生成时间较长，建议在生产环境中使用回调通知而非轮询。</Warning>

<Note>
  HD 版本消耗的积分较多，建议先用标准版或 Turbo 版确认效果后，再使用 HD
  版生成最终成品。
</Note>
