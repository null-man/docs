## 简介

Google Imagen4 Fast 是 Imagen4 的快速版本，在保持较高图像质量的同时大幅提升生成速度，适合需要快速出图的场景。

## 请求示例

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.kie.ai/api/v1/jobs/createTask \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "google/imagen4-fast",
    "callBackUrl": "https://your-domain.com/api/callback",
    "input": {
      "prompt": "一幅电影质感的城市夜景，霓虹灯倒映在雨后的街道上，赛博朋克风格，高对比度",
      "negative_prompt": "",
      "aspect_ratio": "16:9",
      "num_images": "1"
    }
  }'
```

</CodeGroup>

## 参数说明

| 参数                  | 类型   | 必填 | 说明                                    |
| --------------------- | ------ | ---- | --------------------------------------- |
| model                 | string | 是   | 模型名称，固定为 `google/imagen4-fast`  |
| callBackUrl           | string | 否   | 任务完成后的回调地址                    |
| input.prompt          | string | 是   | 图像生成的文字描述                      |
| input.negative_prompt | string | 否   | 负面提示词，描述不希望出现的内容        |
| input.aspect_ratio    | string | 否   | 图像宽高比，如 `1:1`、`16:9`、`9:16` 等 |
| input.num_images      | string | 否   | 生成图像数量                            |

## 与标准版对比

| 特性     | Imagen4  | Imagen4 Fast       |
| -------- | -------- | ------------------ |
| 生成速度 | 标准     | 更快               |
| 图像质量 | 最高     | 高                 |
| 适用场景 | 精细创作 | 快速预览、批量生成 |
