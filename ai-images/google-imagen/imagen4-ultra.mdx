## 简介

Google Imagen4 Ultra 是 Imagen4 系列中的旗舰版本，提供最高质量的图像生成能力，适合对画质有极致要求的专业创作场景。

## 请求示例

<CodeGroup>

```bash cURL
curl --request POST \
  --url https://api.kie.ai/api/v1/jobs/createTask \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "google/imagen4-ultra",
    "callback_url": "https://your-domain.com/api/callback",
    "input": {
      "prompt": "一幅精致的水彩画，描绘春天的樱花树下，花瓣随风飘落，远处是朦胧的山峦，艺术感十足",
      "negative_prompt": "",
      "aspect_ratio": "1:1",
      "seed": ""
    }
  }'
```

</CodeGroup>

## 参数说明

| 参数                  | 类型   | 必填 | 说明                                    |
| --------------------- | ------ | ---- | --------------------------------------- |
| model                 | string | 是   | 模型名称，固定为 `google/imagen4-ultra` |
| callback_url          | string | 否   | 任务完成后的回调地址                    |
| input.prompt          | string | 是   | 图像生成的文字描述                      |
| input.negative_prompt | string | 否   | 负面提示词，描述不希望出现的内容        |
| input.aspect_ratio    | string | 否   | 图像宽高比                              |
| input.seed            | string | 否   | 随机种子，用于复现结果                  |
